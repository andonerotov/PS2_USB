PS2-USB

A PS2 controller to USB adaptor using the PJRC Teensy.

Download this project as a .zip file Download this project as a tar.gz file

What is it?

PS2-USB is exactly what it says on the tin: a simple project which makes use of the PJRC Teensy 2.0 to turn a Sony Playstation 2 controller into a USB HID Joystick.

How does it work?

PS2-USB uses the Teensyduino Joystick library and Bill Porter's PS2X library together, using PS2X to read from the controller, then sending that data to the computer with the Joystick library. It's written in Arduino and requires that Teensyduino is installed.

Will it work with my computer/tablet/sentient robot?

PS2-USB will work with anything that supports the USB HID Joystick protocol, which means that it should work with any modern computer and operating system. Tablets are another matter entirely. I don't think it will work with any Android or iOS devices, but please let me know if it does.

Will it work with my controller?

Possibly. As it stands, the current posted hardware will work with an official Sony Dualshock 2 controller. If you replace the 5.6k resistors with wire jumpers and convert your Teensy to 3.3 volts, more controllers will work. However, PS2x is finicky, so be sure to try everything on this page to get it working. Also try converting your Teensy to 3.3 volts or running it on an external 3.3v supply.

Will it work with my guitar controller?

Eventually. Support for guitar controllers is in the PS2X library, but I have yet to add them to PS2-USB. Once I get everything documented and cleaned up, I'll get cracking on guitar support.

Does it support rumble?

Currently, no. Rumble could be added in a future revision, but I have no idea how to receive force-feedback data from the computer.

How do I set it up?

Check the README file for setup instructions.

Where are the schematics?

Currently, the schematics are in the file "PS2-USB.fzz", made with Fritzing. Soon, I'll have everything drawn up in Cadsoft EAGLE, with pdfs of the schematics and so forth. If you don't want to use Fritzing and can't wait for me to make the schematics, just check the README file for wiring instructions.

Why the Teensy? Why not the Arduino Leonardo/Uno?

I chose the Teensy because it was cheap and easy to use. Dedicating a whole Uno to a simple project like this seemed wasteful, and there are no HID joystick libraries for the Leonardo (that I'm aware of). I'd like to port the Teensyduino libraries (and eventually PS2-USB) to the Leonardo, but that's going to take a while.
